CAROUSEL IMAGES FOR FNA WIFI PORTAL
====================================

Please add 3 images to this folder with the following names:

1. slide1.jpg - Main airport terminal exterior or interior view
2. slide2.jpg - Airport facilities (lounge, shops, etc.)
3. slide3.jpg - Airport amenities or passenger areas

RECOMMENDED IMAGE SPECIFICATIONS:
- Format: JPG or PNG
- Dimensions: 1920x600px (or similar 16:5 ratio)
- File size: Under 500KB each for fast loading
- Quality: High resolution but optimized for web

TIPS:
- Use professional photos of Freetown International Airport
- Ensure images are bright and welcoming
- Images will be displayed with a dark overlay (brightness 70%)
- Horizontal/landscape orientation works best

Once you add your images, the carousel will automatically display them.
